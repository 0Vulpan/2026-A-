# STM32G474 智能喷灌系统
## 2026 吉林省大学生电子设计竞赛 A题 | 二等奖项目

本项目为 2026 年吉林省电子设计竞赛 A 题作品，基于 STM32G474 实现的智能喷灌控制系统，包含完整工程代码与驱动实现。

---

## 项目说明
本仓库仅公开本人在电赛中完成的**完整工程代码**，非完整题目工程，仅作为个人作品归档与学习参考。

涉及模块：
- 主控：STM32G474RCT6
- 库版本：HAL 库
- 串口屏：陶晶驰串口屏（指令交互）
- 传感器：流量传感器采集
- 执行器：
  - 水泵驱动控制
  - PWM 控制舵机
  - 减速电机驱动
  - 线性调光 LED 亮度控制

开发环境：STM32CubeIDE / Keil5
